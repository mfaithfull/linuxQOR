A folder of basic QOR examples.

All examples are platform independent, terminal/console, programs with no dependencies except QOR libraries.

helloworld:-
    A QOR 'Hello World' example. 20 lines of code to get started.

outline:-
    A self analysing app that gives insight into how QOR apps fit
    together.

roller:- 
    Real applications are built from Roles and Features.
    The QOR is designed for large scale applications built
    from these simple parts.

work:- 
    Add workflow state machines to manage the flow of your application.
    Workflows make expressing state management designs like statecharts easy.

options:-
    Add traditional command line options to your application. 

plumbing:-
    Connect to files and the world with QOR Pipelines for IO and data processing.
    Learn how to transcode a file in one statement.

parallel:-
    Launch and use background services and custom parallel tasks. 
    Signals allow notification across threads enabling, for example,
    log aggregation.
