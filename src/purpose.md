contains the QOR source code

/qor - the ubiquitous low level libraries used everywhere, data structures, refelection, memory...
/platform - every bit of code that changes between building for Windows and Linux, Android, Emscripten...
/macros - C PreProcessor headers.
/framework - Application parts and high level flow control, io, threads, pipelines...
/configuration - common configuration headers for building QOR modules
/components - QOR modules built on top of the other libraries. These can be anything, Parsers, Network protocols...
