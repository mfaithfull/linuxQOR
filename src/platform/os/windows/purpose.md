Provides Windows support for the QOR

/api_layer - lazy loaded wrappers and safety checks for the Windows API
/app - current thread and process, dynamic library loading and module bootstraps
/com - COM support - experimental port not used as of QOR 0.5
/common - common Windows types and definitions
/io - Windows IO, Files, Sockets, Async over IO Completion Ports
/ui - Windows UI - experimental as of QOR 0.5
